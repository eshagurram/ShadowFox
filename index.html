Face Detection based Attendance System

This project automates attendance marking using face recognition technology. It uses Python, OpenCV, and the face_recognition library to detect and identify faces in real-time via a webcam. Recognized faces are matched with pre-stored user data, and attendance records are automatically stored in a MongoDB database. The system reduces manual errors, prevents proxy attendance, and ensures accurate and efficient tracking. It demonstrates the practical integration of computer vision and NoSQL databases for real-world applications.
